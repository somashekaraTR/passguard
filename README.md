# PassGuard — Password Strength Analyzer

A browser-based password security tool built with HTML, CSS, and JavaScript.

## Features
- Real-time strength analysis (score 0–100)
- Entropy calculation in bits
- Brute-force crack time estimation
- Complexity checks (length, uppercase, lowercase, digits, symbols)
- Strong password generator
- Session history with SHA-256 hashing

## How to Run
Just open `password-analyzer.html` in any browser — no installation needed.

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Web Crypto API (SHA-256 hashing)
